Joy Timer: Term and Holiday Tracking Device

DATASET
 This device uses custom JSON files, created by the team and uploaded to a Raspberry Pi, to store and manage data for academic terms, holiday schedules for China and the UK, and real-time timekeeping.

DATA DEVICE (Physical)
 The physical component combines two LED strips, an OLED, and a pointer to visualize time. One LED strip represents the academic term's progress, while the other, fixed at the bottom, uses colors to indicate the current term. As time progresses, the lights on the first strip turn off one by one, with the remaining lit LEDs indicating the number of weeks left. The OLED shows the exact names of upcoming holidays, and the pointer supplements the data by visualizing the number of days until the next holiday. This dual-display system provides users with a clear and intuitive way to track both term progress and holiday schedules.

Additionally, users can click a button to reveal more specific deadlines and explore project showcases from previous years' students.

HOW TO
 Use the Unity-based app to sync with the physical device. The app overlays augmented information via AR, enabling users to explore academic progress, view holidays, and interact with 3D models for a seamless planning experience.
