Joy Timer: Term and Holiday Tracking Device

DATASET
 This device uses custom JSON files, created by the team and uploaded to a Raspberry Pi, to store and manage data for academic terms, holiday schedules for China and the UK, and real-time timekeeping.

DATA DEVICE (Physical)
 The physical component combines an LED strip and a pointer to visualize time. The LED strip represents the academic term's progress, with lights turning off as the term advances, while the pointer displays a countdown to upcoming holidays. This dual-display system provides a clear and intuitive way for users to track both term progress and holiday schedules.

DATA DEVICE (Digital)
 The digital dashboard complements the physical device with detailed visualizations sourced from the JSON files on the Raspberry Pi. It includes ring charts for remaining days, reminders for holidays in China and the UK, and access to academic module information. Users can toggle between term-specific and holiday views, and the data dynamically updates for accuracy.

HOW TO
 Use the Unity-based app to sync with the physical device. The app overlays augmented information via AR, enabling users to explore academic progress, view holidays, and interact with 3D models for a seamless planning experience.
